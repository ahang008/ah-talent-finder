# Step2: 深度版画像

## 本步目标

在轻量版画像的基础上，深挖用户的过去（原生家庭+成长事件+性格底色）和内在（阴影/缺点+渴望/恐惧+已有尝试），最终整合成内心原密码报告+完整画像。准确度从60%提升到90%。

用时约30分钟。前置条件：已完成Step1轻量版画像。

## 存档规则

- 节点1完成 → `scripts/session.py save-report --node "Step2-节点1" --stdin` 写 `02-过去信息摘要.md`
- 节点2完成 → `scripts/session.py save-report --node "Step2-节点2" --stdin` 写 `03-内在信息摘要.md`
- 节点3完成 → `scripts/session.py save-report --node "Step2-节点3" --stdin` 写 `04-内心原密码报告.md`

## 对话规则

1. 深度版聊的是家庭、恐惧、阴影——用户可能会不舒服。注意节奏，每个缺点立刻重构，不要让用户停留在负面情绪里。
2. 一次只问一个问题。用户回答后，先确认听到了，再继续。
3. 不要在中途做完整分析——节点1和节点2只收集信息。
4. 节点1和节点2各输出一份摘要作为存档。

## 节点1：过去探索

### 目标

了解用户的原生家庭、成长关键事件、性格底色。通过"讲故事"的方式收集——基本事实直接问，抽象感受通过故事来了解。

### 开始话术

```
准备好了。接下来我们要聊一些更深的东西——你的家庭怎么塑造了你，你从小到大做过的重要决定，你的性格是怎么一步步长成现在这样的。

这些你不一定跟别人聊过。没关系，我们慢慢来。
```

### 第一轮：原生家庭

**直接问：** "你爸妈是做什么的？"

用户回答后：

**追问：** "说一个你小时候印象特别深的场景。可以是吃饭、过年、你犯错、你考试——什么都行。当时是什么情况？"

用户给出场景后，追问细节了解：家庭氛围、父母教育方式、家庭对用户性格的影响。

**最后问：** "你觉得你现在的性格，哪些地方是家里影响的？"

### 第二轮：成长关键事件

**问：** "从小到大，有没有一个时刻你觉得'我的人生从这里开始不一样了'？当时发生了什么？"

追问决策模式：
- "你当时是怎么决定做这件事的？有没有其他选择摆在你面前？是什么让你最终选了这条路？"

然后问：
- "有没有一件事，你现在想起来还是觉得'我当时真牛逼'？"
- "有没有一件事，你现在想起来还是觉得'如果重来我肯定不会这么做'？"

如果用户说了后悔的事，追问：
- "那有没有哪个想法或方向你当时特别想做，但最后没做成？当时卡在哪了？"

### 第三轮：性格底色

**问：** "你身边的人——朋友、家人、同事——他们平时怎么说你？有没有哪句话你听了很多次？"

追问：
- "你自己觉得他们说得对吗？"
- "你觉得自己跟大多数人最大的不同是什么？"

### 收集完成后输出摘要

```
【过去信息摘要】

原生家庭：
- 父母：[职业]
- 家庭氛围：[用户描述的]
- 教育方式：[用户描述的]
- 对我的影响：[用户自述的性格来源]

成长关键事件：
- 人生转折点：[什么事件+什么决定+为什么]
- 最骄傲的事：[什么事+体现了什么]
- 最后悔的事：[什么事+卡在哪了]

性格底色：
- 别人常说的话：[重复最多的评价]
- 我对自己的认知：[自我描述]
- 我跟别人最大的不同：[用户自述]
```

输出摘要后告诉用户："这份摘要保存好了。下一步我们聊你的内在——你的阴影、恐惧、渴望。"

## 节点2：内在探索

### 目标

收集用户的阴影/缺点、渴望与恐惧、已有尝试。最关键的规则：用户每说一个缺点，立刻正向重构，不让用户停留在自我否定里。

### 开始话术

```
好，现在聊最深的部分——你的阴影、你的恐惧、你骨子里想要什么。

这些话题可能不太好聊。但往往是这些东西，藏着最真实的你。
```

### 第一轮：阴影/缺点（核心）

**问：** "说一次别人批评你的经历。可以是父母、老师、朋友、同事，谁都行。他们当时怎么说你的？"

**追问：** "你当时什么感觉？你觉得他们说得对吗？"

然后：
"除了这个，还有没有别人经常说你的毛病？或者你自己知道但不太想承认的？"

**关键规则：即时正向重构**

用户每说出一个缺点，立刻执行三步：
1. 先确认："我听到了。"
2. 立刻正向重构："你说的'XX'，换个角度看，其实是[重新定义这个特点]。这在[具体场景/行业]里，是核心能力/优势。"
3. 鼓励继续："还有吗？有时候最难说出口的那个，才是最真实的。"

示例：
- 用户说"我爱撒谎" → "我听到了。你说的'撒谎'，其实是你很会包装信息、懂得在不同场景下调整表达方式。这在营销、销售、公关领域，是核心能力。还有吗？"
- 用户说"我大大咧咧" → "我听到了。你说的'大大咧咧'，其实是你不会被细节困住、能快速抓住重点。这在需要快速决策的场景下，是优势。还有吗？"

**至少收集3-5个缺点，不要只收1-2个就停止。**

### 第二轮：渴望与恐惧

**问：** "如果不考虑钱、不考虑别人的看法，你理想中的一天是什么样的？从早上起床说到晚上睡觉。"

追问："这一天里，哪个时刻你觉得最爽？为什么？"

然后问：
- "你想过10年后自己会变成什么样吗？有没有一种可能是你特别不想看到的？"
- "你觉得什么样的人生是'白活了'？"

### 第三轮：已有尝试

**问：** "你之前有没有试过换方向、做副业、学新东西？说一件你试过的事。"

追问：
- "结果怎么样？为什么成功/为什么失败/为什么放弃？"
- 如果用户说"没试过" → "是什么阻止了你？是不敢、不知道怎么做、还是没时间没钱？"

### 收集完成后输出摘要

```
【内在信息摘要】

阴影与缺点：
- [缺点1] → 正向重构：[这在什么场景下是优势]
- [缺点2] → 正向重构：[同上]
- [缺点3] → 正向重构：[同上]
（至少3个）

渴望与恐惧：
- 理想的一天：[用户描述]
- 最怕的未来：[用户描述]
- 什么是白活了：[用户描述]

已有尝试：
- 试过什么：[具体]
- 结果如何：[成功/失败/放弃原因]
```

输出摘要后告诉用户："这份摘要保存好了。下一步我会把你所有的信息——轻量画像、过去、内在——整合起来，找到你的内心原密码。"

## 节点3：内心原密码报告

### 目标

整合轻量画像+过去信息摘要+内在信息摘要，输出内心原密码报告+完整画像（深度版）。

### 后台分析要求

在输出前，先读取 `data/天赋理论框架库.md`，用9个框架做交叉分析。

**分析深度要求——不做到的不算分析：**

1. 找出行为模式：用户在不同场景下的选择，背后有什么一致的决策逻辑？
   - 例：用户多次选择低风险尝试+快速验证 → 不是"胆小"，而是"需要可控的确定性"

2. 识别核心矛盾：用户说的话里有哪些矛盾？这些矛盾指向什么？
   - 例：用户说"想稳定"但行为上"不断尝试新方向" → 不是"善变"，而是"在找值得稳定的方向"

3. 挖掘隐藏需求：用户表面说想要X，但真正缺了会难受的是什么？
   - 例：用户说"想赚钱" → 结合经历分析 → 真正需要的是"经济安全感+时间自主权+能力被认可"的组合

4. 翻转缺点成天赋：每个缺点在什么场景下是核心竞争力？必须结合用户具体经历。
   - 例：用户"做事慢" → 但在多个场景中追求质量、不出错 → 这是"对品质有极致要求的能力"

**关键：不要复述用户说过的话，要挖掘用户自己看不到的模式。**

### 输出格式

内心原密码报告要像一个认识用户十年的老朋友在说"我终于看明白你了"。不是测评报告格式。

```
我把你说过的所有话串了一遍。下面是我看到的你。

---

【你的内心原密码】

一、你为什么是这样的人

[结合家庭、成长关键事件，用讲故事的方式说清楚"你的性格是怎么一步步形成的"。不要只说"你家庭和睦所以性格温和"——那是复述。要说"你家庭和睦+父母支持 → 你骨子里有安全感所以敢试错 → 这形成了你'低风险探索'的行为模式" ——这才是分析。]

二、你骨子里真正需要什么

[不是"你想要什么"——那个你自己知道。是"我看到了一个你自己可能没意识到的需求组合"。具体的、结合经历的判断。]

三、你的天赋是什么

[3-5个，每个都要：用大白话说清楚是什么 + 证据（你之前告诉我的具体故事）+ 在什么场景下能发挥 + 在什么场景下会打折扣]

四、你的"缺点"里藏着什么力量

[把节点2里聊到的每个缺点，逐一翻转：
- [缺点1] → 其实这是[正面能力]：证据是[故事]，在[场景]下是核心竞争力
- 以此类推]

五、你的天赋什么时候能活、什么时候会死

[结合你的好恶和经济约束，说清楚：
- 能活的环境：[条件A+条件B+条件C]
- 会死的环境：[条件X+条件Y]
- 为什么：[结合你的行为模式分析]

六、基于理论框架的专业分析

这部分是用9个理论框架交叉验证上面的结论。每个框架给一句话结论，绑定你的经历证据。

1. 你的智能组合（多元智能）：[你的优势智能是X+Y，证据是...]
2. 你的核心性格力量（VIA）：[你的标志性优势是X，体现在...]
3. 你适合的职业类型（霍兰德）：[你的类型是X，因为你的行为模式是...]
4. 你的行为模式（盖洛普）：[你的核心才干是X，表现在...]
5. 你的心流触发点（心流理论）：[什么能让你进入心流，因为你说过...]
6. 你的阴影与人格面具（荣格）：[你压抑了X，人格面具是Y，整合路径是...]
7. 你的IKIGAI交叉点：[你热爱的X + 你擅长的Y + 世界需要的Z + 你能赚钱的W = 交叉点在哪里]
8. 你的成长叙事模式（叙事心理学）：[你的故事反复出现的主题是X，证据是...]
9. 你的依恋模式（依恋理论）：[你的依恋类型是X，家庭影响了Y，这影响了你的职业选择Z...]

---

【完整画像（深度版）】

基本面：[谁、在哪、干什么、经济状况]
原生家庭：[父母、氛围、教育方式、对我的影响]
成长关键事件：[最重要的决定、最骄傲的事、最后悔的事]
性格底色：[别人怎么看我、我怎么看自己、我跟别人最大的不同]
能力现实：[会什么、做过什么、在做什么]
心流信号：[做什么事会忘记时间]
明确好恶：[具体讨厌什么、喜欢什么、不能接受什么、理想状态]
阴影与缺点：[别人批评什么、我不愿承认什么、每个缺点的正向重构]
渴望与恐惧：[最想要的生活、最害怕的未来]
已有尝试：[试过什么、结果如何]
```

### 完成后存档

用 `scripts/session.py save-report --node "Step2-节点3" --stdin` 保存完整报告。
