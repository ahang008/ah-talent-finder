# Step3: 深度假设精炼

## 本步目标

路径A用户完成Step2深度版画像后，把深挖到的家庭、阴影、渴望等信息，跟Step1的初步假设做对比——哪些假设被强化了、哪些需要调整、哪些新的假设浮现了。最终输出一份更精准的天赋假设和验证实验。

## 前置条件

- 必须已有 `01-轻量画像.md`（Step1初步假设+实验）
- 必须已有 `04-内心原密码报告.md`（Step2深度版画像）
- 路径B和路径C用户不经过此步

## 存档规则

- 精炼分析完成 → `scripts/session.py save-report --node "Step3-方向推荐" --stdin` 写 `05-深度假设对比.md`
- 更新实验完成 → `scripts/session.py save-report --node "Step3-验证方案" --stdin` 写 `06-深度验证方案.md`

---

## 第一部分：假设对比

### 分析流程

1. 读取 `01-轻量画像.md`，提取Step1中的几条天赋假设
2. 读取 `04-内心原密码报告.md`，提取深度分析中的核心发现
3. 逐条对比：

**对每条Step1假设，问：**
- 深度版的信息是强化了这条假设，还是动摇了它？
- 有没有新的证据支持它？（特别是家庭影响、成长关键事件中的证据）
- 有没有新的证据质疑它？（特别是阴影/恐惧/渴望中的矛盾）
- 这条假设需要修正吗？修正后是什么？

**然后问：**
- 深度版里有没有浮现Step1完全没看到的假设？
- 用户的"缺点翻转"是不是指向了新的天赋方向？
- 用户的恐惧和渴望是不是解释了为什么某些天赋一直被压抑？

### 输出格式

自由叙事，不用板块标题。必须覆盖：
- Step1假设逐条回顾 + 深度版证据对照
- 被强化的假设（哪些、为什么）
- 需要调整的假设（原来怎么想、现在怎么想、为什么变）
- 新浮现的假设（深度版挖出来的）
- 综合后的天赋假设（3-5条，每条有深度证据）

```
好，我们聊完你的家庭、童年和内心那些东西之后，我回头看了一眼之前那几条假设。

有些更稳了，有些需要修正，有些新的东西浮现了。

---

[自由叙事——逐条对照，说明变化和原因]

---

综合以上，结合你的表面经历和深层模式，这几条假设目前最值得验证：

[精炼后的假设，每条带深度证据+反证]
```

## 第二部分：更新验证实验

### 实验设计原则

1. 基于精炼后的假设设计实验
2. 不需要辞职，2周内可完成，最好0成本
3. 每个动作具体到"做什么/找谁/怎么判断"
4. 必须有明确的继续/放弃信号
5. 深度版中发现的"天赋能活/会死的环境条件"，必须在实验设计中体现——设计一个让天赋"能活"的小环境来测试

### 输出格式

```
【14天验证实验 v2】

[基于深度分析更新后的实验设计]

为假设X设计的验证：
- 具体做什么
- 观察什么
- 继续的信号
- 放弃的信号

...

48小时内第一步：
[一个具体的动作]

14天后你回来告诉我。到那时候你自己就能判断——不需要我。
```
