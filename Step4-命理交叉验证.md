# Step4: 命理交叉验证（选修）

## 本步目标

用古典命理典籍（穷通宝典、滴天髓、三命通会等）从出生时间推导先天倾向，与AI天赋分析的结论做两套独立系统的交叉验证。如果两套系统指向同一结论，该结论更稳。

## 前置条件

- 已完成Step1轻量版画像（至少03-职业方向清单）
- 用户提供：出生年月日时（精确到小时）、出生地、性别

## 重要说明

这一步不是迷信，是两套独立系统交叉验证：
- 心理学框架从用户的真实经历里找规律
- 命理从用户的出生时间推导先天倾向
- 两套系统互不认识，如果它们指向同一个结论，这个结论就更稳
- 有差异也是正常的——差异本身提供信息

如果用户不知道出生时间、不方便提供、或者不信命理——跳过这一步，不会影响前面的结论。

---

## 第一步：收集出生信息

如果用户还没提供，先问：

```
你想做命理交叉验证的话，需要几个信息：
- 你的出生年月日时（阳历/公历，精确到小时）
- 你的出生地（省+市）
- 你的性别

如果不知道具体时辰、或者不方便说，这一步可以跳过的。
```

## 第二步：排盘

使用 `mcp__bazi__get_bazi_details` 工具，传入用户提供的出生年月日时，获取八字排盘结果。

在排盘结果基础上，提取：
- 四柱（年柱、月柱、日柱、时柱）
- 日主天干
- 十神分布
- 五行分布

展示给用户：

```
先看你的八字命盘：

年柱：[干支]
月柱：[干支]
日柱：[干支]
日主：[X]（你的核心属性）
时柱：[干支]

主要十神：[列出最突出的3-5个十神]
五行分布：[简要说明]

（如果你想了解命盘的任何术语，随时问我——我会用大白话解释。）
```

## 第三步：古典命理天赋分析

基于排盘结果，从五部典籍中提取与天赋/先天倾向/适合方向相关的内容。每个分析都必须引用用户的具体命盘信息。

### 1. 穷通宝鉴——日主性情

根据用户的日主天干和月令：
- 这部经典如何描述这个日主的先天性情和天赋倾向？
- 结合月令，日主在什么条件下最能发挥？
- 对应的现代天赋是什么？

### 2. 滴天髓——天干之道

- 用户的日主天干在《滴天髓》中的"道"是什么？
- 这意味着核心天赋指向什么方向？
- 翻译成现代能力语言

### 3. 三命通会——十神与职业倾向

- 用户的十神组合（特别是食神、伤官、正印、偏印、正官、七杀、正财、偏财）指向什么类型的职业/角色？
- 哪些十神是用户天赋的核心？为什么？
- 翻译参考：
  - 食神 → 表达天赋、创作能力、教学能力、审美能力
  - 伤官 → 创新天赋、技术能力、突破常规的思维
  - 正印 → 学习天赋、研究能力、知识整合能力
  - 偏印 → 特殊才能、玄学/技术天赋、非传统思维能力
  - 正官 → 管理天赋、规则制定能力、组织能力
  - 七杀 → 决断力、竞争天赋、开拓能力
  - 正财 → 经营天赋、积累能力、价值判断力
  - 偏财 → 商业嗅觉、投资眼光、资源整合能力
  - 比肩 → 执行天赋、竞争中的稳定性
  - 劫财 → 行动力、社交天赋、开拓精神

### 4. 渊海子平——格局与方向

- 用户的格局是什么？
- 这个格局的人通常往哪个方向发展？
- 用户的用神是什么？用神指向什么天赋？

### 5. 神峰通考——病药论

- 用户的命局中"病"在哪里（什么在阻碍天赋发挥）？
- "药"是什么（如何激活天赋）？

## 第五步：交叉验证

将命理分析结果与前面Step1/Step2/Step3中拿到的AI天赋分析结论做对比：

```
【交叉验证】

以下结论两套系统一致：
1. [天赋/方向] —— AI分析的证据是[用户故事]，命理的证据是[十神/格局]，两套独立系统指向同一个方向
2. [天赋/方向] —— ...

以下结论有差异：
1. [具体差异] —— AI分析说[X]，命理说[Y]
   可能的原因：[分析]
   建议：[以哪个为准或如何调和]

命理补充了AI分析没有看到的角度：
1. [角度] —— [说明]

综合结论：
基于两套系统，你最应该关注的天赋方向是：
1. [方向] —— [两套系统的综合证据]
2. [方向] —— [同上]
3. [方向] —— [同上]
```

### 交叉验证原则

1. 一致的结论：增强信心，重点关注
2. 有差异的地方：不急着否定任何一个，差异本身就是信息——可能AI分析偏重"后天经历形成的天赋"，命理指向"先天倾向但还没发展出来的天赋"
3. 不要用命理否定AI分析的结论，也不要用AI分析否定命理——它们是互补的
4. 用大白话输出，每个古典术语第一次出现时必须解释
